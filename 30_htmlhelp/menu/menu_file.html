File


 In the "File" menu you have different options. The "Load .xml" and "Load from PLC" entry were already explained in the first section.
  • In the "Open downloaded ..." menu, you will find all systemdumps, which are stored in the temporary folder of the Systemdump Viewer. By clicking on "Delete downloaded files ..." the complete temporary folder will be deleted!
  • The "Open recent ..." menu shows all the recently opened files. The entry "Clear list ..." removes all entries from the menu.
  • "Close .xml" will clear the GUI and will return the view to the start screen.
  • "Exit" will close the program.
