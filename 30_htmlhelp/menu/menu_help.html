Help

From the help menu you can:

  • call the help for the Systemdump Viewer
  • show the "About" dialog
  • check for software updates

The update server can be set in the options. Normally this entry can be left blank, so the default server is used. Please note that this is no commercial software and it could be that the server is offline from time to time.
