Export

In the "Export" menu you can save the different files to your computer. If you have enabled the "Download data objects" feature in the options and used the internal "Download from PLC" feature. You can save every data object as single file, or all files together as .zip.

Also you can export a pdf report, which prints the information from the Systemdump into pdf format. In the actual version the only report type is a hardware report, containing the following information:

  • Hardware name
  • Revision
  • Firmware
  • Serial number
