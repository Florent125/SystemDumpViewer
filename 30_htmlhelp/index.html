Systemdump Viewer

Note:
Please notice, that the SystemdumpViewer is NOT an official B&R product. Therefore there is no guaranteed support or maintenance on this tool. However, if you have any ideas for future improvements or should run into bugs, you're welcome to write me an email.

The SystemdumpViewer is a little tool, that makes it easier for you to take a look at and to analyse the contents of a Systemdump.xml file. In its second version the funcionality was extended, and the GUI was partly redesigned. If you don't want to read the whole manual but still want to get a few useful hints, take a look at this page.

Features:

  1. Opening Systemdumps
    • Open Systemdump.xml files from disk
    • Open Systemdump directly from PLC
    • Recent / downloaded files
  2. Export functions
    • Export single files to disk
    • Export all files to disk
    • Create reports
  3. Analyzing the Systemdump.xml
    • Links directly to the B&R homepage
    • Communication to the B&R online help from the Automation Studio
