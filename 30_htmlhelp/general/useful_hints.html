Useful hints

If you should experience any bugs please help me to increase the software quality by sending a small description of the bug and the corresponding Systemdump.xml file to this address.

Also if you have any ideas which improve the usability or the functionality of this software, I'd be happy if you would let me know!

Opening a Systemdump

  • Drag & drop Systemdump.xml files on the main screen
  • Open Systemdump.xml from the windows context menu "Open with ..." Systemdump Viewer
  • Download a Systemdump from a PLC by entering the IP address

Exploring a Systemdump

  • Click on a section on the main screen to jump directly to the right window
  • Press "F1" key on hardware or logger entry to open it in B&R help
  • Right click on hardware modules to see what action possibilities you have
  • Resizable windows and table columns for better readability

Export functions

  • Save the downloaded Systemdump.xml file (downloaded from PLC) on the computer
  • Save the downloaded data objects (Network Command Trace, Logger files, Profiler objects ...)
  • Create PDF reports from the Systemdump contents
