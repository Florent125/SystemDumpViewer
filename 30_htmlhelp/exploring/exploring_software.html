Software

In the "Software" view you can browse through the differen Application Modules and in addition, you can filter the table by object type:

  • All objects
  • Cyclic objects
  • Data objects
  • Motion objects
  • Config objects
  • Other objects

In this way, you can see all the tasks and objects that are really installed on the target.
