Logger


For each logger that is included into the actual Systemdump.xml file, you will get a checkbox. Every logger that is selected will be added to the table. You have the following possibilities:
  • Select multiple logger modules
  • Sort the table by clicking the column headers
  • Open a logger entry in the B&R help by selecting it and pressing the F1 key
  • Resize the window and the columns for better view
